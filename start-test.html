Это тест для проверки зрения по новому методу, который был разработан группой врачей-офтальмологов

Инструкция по прохождению теста:
1. Сядьте на расстояние 40-50см. от экрана.
2. Голова должна быть зафиксирована строго по центру монитора.
3. Включите звук(оденьте наушники).
В процессе проверки будут издаваться короткие звуки для улучшения реакции.
4. После нажатия кнопки «START» блок с текстом начнет увеличиваться.
В блоке 3 строки, размер шрифта которых уменьшается сверху вниз.
5. Как только вы смогли четко прочитать последнюю строку – нажмите кнопку «STOP»
